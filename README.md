# Taskify

Taskify, görev yönetimini kolaylaştıran modern bir web uygulamasıdır. Kullanıcılar projelerini oluşturabilir, görevleri düzenleyebilir ve durumlarını takip edebilirler. Bu frontend, React ile geliştirilmiştir ve REST API üzerinden backend ile iletişim kurar.

## Özellikler

- **Görev Yönetimi:** Projeler oluşturun, görevleri ekleyin ve durumlarını güncelleyin.
- **Kullanıcı Profili:** Kullanıcılar profil bilgilerini görüntüleyebilir ve düzenleyebilir.
- **Responsive Tasarım:** Mobil ve masaüstü cihazlarda uyumlu kullanıcı arayüzü.
- **JWT Kimlik Doğrulama:** Güvenli giriş ve kullanıcı doğrulama işlemleri.
- **Şifre Değiştirme:** Kullanıcılar mevcut şifrelerini değiştirerek güvenliklerini artırabilirler.

## Link
https://taskify.rumer.tr/

### Kullanım

- Kullanıcılar giriş yaptıktan sonra projelerini görüntüleyebilir ve yönetebilirler.

- Yeni görevler ekleyebilir, mevcut görevleri düzenleyebilir veya silebilirler.

- Profil sayfasında kişisel bilgilerini güncelleyebilirler.
